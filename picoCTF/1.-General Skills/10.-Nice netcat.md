## Nice netcat
# Descripción
There is a nice program that you can talk to by using this command in a shell:$ nc wily-courier.picoctf.net 52410, but it doesn't speak English...

# Solución
## Solucion 1
El comando funciona de la siguiente manera: 
- **`nc ...`**: Connects to the server and receives the stream of numbers.
    
- **`sys.stdin.read().split()`**: Takes all those numbers and puts them into a list.
    
- **`chr(int(x))`**: This is the magic part. The function $chr()$ takes a decimal integer and converts it to its corresponding ASCII character.
    
    - Example: $chr(112) \rightarrow 'p'$
        
    - Example: $chr(105) \rightarrow 'i'$
        
- **`''.join(...)`**: Glues all those characters together to reveal your `picoCTF{...}` flag.
```

jorge_moran-picoctf@webshell:~$ nc wily-courier.picoctf.net 52410 | python3 -c "import sys; print(''.join(chr(int(x)) for x in sys.stdin.read().split()))"

```
