## Lets Warm Up
# Descripción
If I told you a word started with 0x70 in hexadecimal, what would it start with in ASCII?

# Solución
## Solucion 1
- Usando Cyberchef
- https://gchq.github.io/CyberChef/#recipe=From_Hex('0x')&input=MHg3MA
## Solución 2
- Usando Python 
```
- >>> int(0x70)
112
>>> chr(112)
'p'
>>> ord('p')
112
>>> 
```
