# Solución
## Solucion 1
- Usando la terminal de PicoCTF y obteniendo el archivo mediante wget, posteriormente dandole permisos y siguiento los pasos indicados como se observa en la terminal.
```
jorge_moran-picoctf@webshell:~$ file static

jorge_moran-picoctf@webshell:~$ file ltdis.sh
ltdis.sh: Bourne-Again shell script, ASCII text executable
jorge_moran-picoctf@webshell:~$ chmod +x ltdis.sh
```
