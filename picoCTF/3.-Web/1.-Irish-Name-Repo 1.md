# Notas Adicionales 
- **`-X POST`**: Enviamos los datos como un formulario (que es como lo hace el navegador).
    
- **`-d "username=' OR 1=1 --"`**: Este es el "payload". La comilla `'` cierra la cadena de texto en la base de datos, y `OR 1=1` hace que la validación sea siempre exitosa. El `--` comenta el resto de la consulta original para que no cause errores.
    
- **`password=password`**: El valor de la contraseña no importa porque el `--` en el usuario hará que el servidor la ignore.
