# PW crack 1
# Descripción
Can you crack the password to get the flag?Download the password checker [here](https://artifacts.picoctf.net/c/12/level1.py) and you'll need the encrypted [flag](https://artifacts.picoctf.net/c/12/level1.flag.txt.enc) in the same directory too.

# Solución
```
jorge_moran-picoctf@webshell:~$ wget -q https://artifacts.picoctf.net/c/12/level1.py
jorge_moran-picoctf@webshell:~$ wget -q https://artifacts.picoctf.net/c/12/level1.flag.txt.enc
jorge_moran-picoctf@webshell:~$ python level1.py
Please enter correct password for flag: hola
That password is incorrect
jorge_moran-picoctf@webshell:~$ nano level1.py
jorge_moran-picoctf@webshell:~$ python level1.py
Please enter correct password for flag: 8713
Welcome back... your flag, user:
picoCTF{545h_r1ng1ng_1b2fd683}
```

# Notas Adicionales 
- Se accede mediante nano al archivo .py para poder ver la contraseña
