# PW crack 2
# Descripción
Can you crack the password to get the flag?Download the password checker [here](https://artifacts.picoctf.net/c/13/level2.py) and you'll need the encrypted [flag](https://artifacts.picoctf.net/c/13/level2.flag.txt.enc) in the same directory too.

# Solución
```
jorge_moran-picoctf@webshell:~$ wget -q https://artifacts.picoctf.net/c/13/level2.py
jorge_moran-picoctf@webshell:~$ wget -q https://artifacts.picoctf.net/c/13/level2.flag.txt.enc
jorge_moran-picoctf@webshell:~$ python level2.py
Please enter correct password for flag: 3455
That password is incorrect
jorge_moran-picoctf@webshell:~$ nano level2.py
jorge_moran-picoctf@webshell:~$ python level2.py
de76
Please enter correct password for flag: de76
Welcome back... your flag, user:
picoCTF{tr45h_51ng1ng_489dea9a}
```

# Notas Adicionales 
- Se accede mediante nano al archivo .py y se modifica para que nos de la contraseña previo a pedirnos la contraseña
